## Курсовая работа на тему: "Проектирование и реализация распределённой системы контроля доступа на базе RFID". 
RFID - технология беспроводной радиочастотной идентификации меток (ключей, карт). Используется для создания электронных замков по пропускам.
Система для управления задачами с использованием Java, Spring, PostgreSQL.
Система обеспечивает создание и удаление ключей доступа к замкам, а также проверку подходит ли ключ к замку.
Реализовано только API. Доступ к API есть только у замков и администраторам по управлению доступом.

## Описание:
1. Сервис поддерживает аутентификацию и авторизацию пользователей по логину и паролю.
2. Доступ к API аутентифицирован с помощью JWT токена.
3. Администратор может управлять всеми замками и ключами: добавлять и удалять ключи и замки, а также предоставлять доступ к ключу к замку
4. Замок может проверят, подходит ли ключ к замку, а также добавлять и удалять ключи 
5. Присутствуют unit тесты и интеграционные с использование test containers
